Pre nego sto pocnemo 
Upozorenje folder Neki RTS Valjda je mnogo velik ali bas mnogo 
ako ne zelite da skidate ceo projekat ili samo brzo da pregledate kod koji sam ja kucao
nalazi se u: Neki RTS valjda -> Assets -> Scripts

Kako bi trebalo otvoriti ovo cudo od programa.
mislim da je najlakse preko Unity Hub nisam ni siguran da li postoji neki drugi nacin
1. Otvoriti Unity Hub
2. add
3 naci "neki RTS Valjda"
4 kraj. Usli ste u Igircu

ako zeliti da pokrenete igricu trebalo bi negde da bude "|>" ovako nekakvo dugme

ako pokrenete samo sln otvorice samo deo programa tj kod koji sam ja kucao.
